# Eclipse Handbook Antora Prototype

This project demonstrates a prototype for restructuring the Eclipse Project Handbook using Antora.

## Features

* Multi-page documentation
* Sidebar navigation
* Cross-references between chapters
* Antora-based build system

## Goal

To convert the existing single-page handbook into a scalable, modular documentation system.

## Preview

Run:
antora antora-playbook.yml

Then open:
build/site/index.html
